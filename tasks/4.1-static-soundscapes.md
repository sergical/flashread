# Task 4.1: Static Soundscapes

**Status:** pending  
**Phase:** 4 - Audio Features  
**Depends on:** Task 1.7  
**Estimated Time:** 45 minutes

## Goal

Add background music/soundscapes during reading.

## Steps

### 1. Source Audio Tracks
Royalty-free sources:
- Pixabay
- Free Music Archive
- Uppbeat

Categories to include:
- [ ] Lo-fi beats (2-3 tracks)
- [ ] Nature sounds: rain, forest, ocean (3 tracks)
- [ ] Ambient/drone (2 tracks)
- [ ] White/pink noise (2 tracks)
- [ ] Focus/study music (2 tracks)

**Total: 10-15 tracks, ~50-100MB**

### 2. Audio Hosting
Options:
- [ ] Bundle with extension (increases size)
- [ ] Host on CDN (Cloudflare R2, S3)
- [ ] Stream from public URLs

### 3. Audio Player Component
- [ ] Add to overlay UI
- [ ] Play/pause button
- [ ] Volume slider
- [ ] Track selector dropdown

### 4. Settings
```typescript
interface AudioSettings {
  enabled: boolean;
  category: 'lofi' | 'nature' | 'ambient' | 'noise' | 'focus';
  track: string;  // track ID
  volume: number; // 0-100
}
```

- [ ] Add to options page
- [ ] Persist in settings sync

### 5. Playback Logic
- [ ] Auto-play when reading starts (if enabled)
- [ ] Loop tracks seamlessly
- [ ] Pause when reading pauses
- [ ] Stop when reading ends
- [ ] Resume on continue
